# Real-Time-Gesture-Typing
First Computer Vision project

## Objective
Write text using real-time hand gestures (A-Z + delete + space)

## Solution
Train a computer vision model on a hand gesture dataset and use it for real-time prediction via webcam.

## Features
- Recognize A-Z gestures
- Space and delete gestures
- Real-time prediction
- On-screen text output

## Requirements
hand_landmarker.task -> command: curl -o hand_landmarker.task https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/1/hand_landmarker.task
## Dataset
This project uses a hand gesture dataset from Kaggle:

- Dataset: [ASL Alphabet]([https://www.kaggle.com/SEU-LINK-AQUI](https://www.kaggle.com/datasets/grassknoted/asl-alphabet))
- Source: Kaggle

Download the dataset from Kaggle and place it in the `data/` folder.
